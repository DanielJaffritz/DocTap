This is a [Next.js](https://nextjs.org) project, named DocTap. Here you can edit and collaborate documents with your team

## Getting Started

First, run the development server:

```bash
npm run dev:all
# or
yarn dev:all
# or
pnpm dev:all
# or
bun dev:all
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
